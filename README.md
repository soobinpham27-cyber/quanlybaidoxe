Hệ Thống Quản Lý Bãi Đỗ Xe Thông Minh (Hybrid Cloud)Dự án này là hệ thống quản lý bãi đỗ xe toàn diện, kết hợp sức mạnh của các dịch vụ đám mây hiện đại để đảm bảo khả năng hoạt động 24/7 và tính bảo mật cao.🏗️ Kiến trúc Hệ thống (Tech Stack)Hệ thống được xây dựng trên mô hình Hybrid Cloud, tận dụng tối đa các dịch vụ miễn phí tốt nhất hiện nay:Backend: Node.js & Express chạy trên Render Cloud.Database Realtime: Firebase Realtime Database dùng để điều phối hàng đợi (Queue) và cập nhật sơ đồ bãi xe tức thời.Database SQL: MySQL (Aiven Cloud) đặt tại Singapore, dùng để lưu trữ lịch sử giao dịch vĩnh viễn.Frontend: Giao diện Web (HTML/JS/CSS) được tối ưu hóa cho cả người quản lý và khách hàng.🌟 Tính năng Nổi bậtĐiều hướng tự động: Hệ thống tự động tính toán và cấp ô trống cho xe khi vào cổng.Giám sát thời gian thực: Sơ đồ bãi xe cập nhật trạng thái (Trống/Có xe) ngay lập tức trên mọi thiết bị.Quản lý lịch sử: Lưu trữ chi tiết mã giao dịch, vị trí đỗ và thời gian chuẩn xác vào MySQL Cloud.Bộ lọc nâng cao: Hỗ trợ tìm kiếm ô số và lọc lịch sử giao dịch theo hành động, vị trí hoặc khoảng thời gian.Hoạt động độc lập: Hệ thống chạy hoàn toàn trên Cloud, không phụ thuộc vào máy tính cá nhân.🛠️ Cấu hình Biến môi trường (Environment Variables)Để đảm bảo bảo mật, các thông tin nhạy cảm được cấu hình thông qua biến môi trường trên Render thay vì lưu trực tiếp trong mã nguồn:BiếnMô tảFIREBASE_CONFIGNội dung JSON của file chìa khóa Firebase.PORTCổng dịch vụ do Render cấp phát (Mặc định: 10000).🚀 Hướng dẫn Chạy Dự ánClone mã nguồn: Tải toàn bộ file từ GitHub Repository này.Cài đặt thư viện: Chạy lệnh npm install để cài đặt các module cần thiết (express, mysql2, firebase-admin, cors).Khởi động Cloud Server: * Upload mã nguồn lên GitHub.Kết nối GitHub với Render.com và thiết lập các biến môi trường cần thiết.Truy cập giao diện: Mở file index.html (Cổng xe) hoặc he_thong.html (Admin) và cập nhật đường dẫn API tới link Render của bạn.
